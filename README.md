# Angularjs Chart Directives

SVG Chart Directives Built for Angularjs.

#### Chart Types
- [x] Bar Chart
- [ ] Line Chart
- [ ] Pie Chart


## Getting Started

#### Requirements
- Angularjs
- That's it!
```
<bar-chart chart-data="data"></bar-chart>
```
All other attributes are optional.
